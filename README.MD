This Nifty Option Buying strategy operates entirely on 1-minute charts and uses the Nifty index for strike selection while executing trades in options. At market open, the close of the first 1-minute Nifty candle is taken as ATM and used as the strike for both CE and PE, which are traded independently. For each option, the close of its first 1-minute candle is marked as a reference line. A buy is triggered when a candle closes above this line, the average price is above the line, and the average price is below the candle close, upon which one lot is bought. The position is exited if any candle closes below the marked line, and re-entries are allowed when conditions match again, with the lot size increasing by one on every new trade independently for CE and PE. Universal exits apply where all open positions are closed and no further trades are taken if the combined CE+PE profit reaches 50 points or when the time hits 15:20.
